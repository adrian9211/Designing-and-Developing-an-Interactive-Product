# Task 5
1


Create a php script that checks to see if a user has a winning lottery ticket.
The user must enter the day the lottery draw takes place - Lucky Lottery draws only take place on Wednesdays and Saturdays.
If they enter a day that is not a Wednesday or a Saturday, they receive a message – ‘The lottery draw only takes place on either a Saturday or a Wednesday’.
The user must also enter what colour their ticket is and what the four digit number on their ticket is.
Both the colour and the number must match for the ticket to be considered a winner. You can hard code the winning colour and ticket number in to separate variables.
